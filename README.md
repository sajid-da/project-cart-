## Repository Structure

This repository is organized cleanly for the university project submission. Below is a description of the contents of each folder:

* **Working_Code/**: Contains all source code files for the Retail Intelligence Suite application (frontend, backend, database config, etc.).
* **Project_Report/**: Upload the final project report here in PDF format.
* **Final_PPT/**: Upload the final presentation file here.
* **Published_Paper/**: Upload the conference or journal paper here (if available).
* **Attendance_Sheet/**: Upload the scanned attendance sheet here.
* **GitHub_Link_Info/**: Contains information regarding the uploaded repository.

## How to use
This branch structure serves as an organized medium to upload the final documents. To run the application, navigate to `Working_Code/` directory and execute the standard startup commands.
