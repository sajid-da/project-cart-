## Required Changes

### 1. Get Real User Location (MANDATORY)

Use browser geolocation:

```js
navigator.geolocation.getCurrentPosition((pos) => {
  const { latitude, longitude } = pos.coords;
});
```

Store as:

```js
const userGPS = { lat, lng };
```

---

### 2. Define Warehouse Anchor Location

Choose a fixed real-world location (example: a Walmart store or any coordinates):

```js
const storeLocation = {
  lat: 12.9716,
  lng: 77.5946
};
```

This acts as the **origin of your 3D map**

---

### 3. Convert GPS → 3D Coordinates

Map real-world coordinates into your 3D space:

```js
const convertTo3D = (lat, lng) => {
  const scale = 10000; // adjust for sensitivity

  return [
    (lng - storeLocation.lng) * scale,
    0,
    (lat - storeLocation.lat) * scale
  ];
};
```

* This makes real GPS behave like movement inside your 3D warehouse

---

### 4. Sync User Position

* Convert user GPS → 3D position
* Update “You are here” marker in real-time

```js
const userPosition3D = convertTo3D(userGPS.lat, userGPS.lng);
```
